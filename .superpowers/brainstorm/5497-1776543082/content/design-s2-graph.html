Design: Section 2 — Graph Visualization

Node color = entry type. Node size = connection density. Edge thickness = co-retrieval strength.

Color Legend — Each Type Gets a Distinct Color
Ghost Knowledge — purple, always large
Structural / Module — blue
MD Document — cyan (plans, specs, ADRs)
Learning — green (harvested sessions)
Error Pattern — red
Convention — amber
Decision / ADR — yellow
Graph View — Mockup (node size = connection count)

What changes from current graph view

  • Node color = entry type (7 distinct colors, not confidence shades)
  • Node size = number of connections (centrality = bigger node)
  • Edge thickness = co-retrieval strength (thin = structural link, thick = frequently co-retrieved)
  • MD doc nodes appear as first-class citizens (cyan) — click to open the MD file in the Docs tab
  • Ghost nodes always render large + purple regardless of size formula
